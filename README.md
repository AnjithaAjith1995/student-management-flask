# Student Management System (Flask + SQLite)

A simple web-based Student Management System built using Flask and SQLite.

## Features
- Add student
- View students
- Edit student details
- Delete student
- Login system (basic session authentication)

## Tech Stack
- Python
- Flask
- SQLite
- HTML / CSS
